To run this project--
Firstly you need to create orclpdb database, then run the sql files provided in the sql folder.

Secondly, you need to copy css, js, images, package_images inside public_allpageand html folder.
Then make sure you have installed orcledb, express packages in your node.js
Finally, run the code by command node src\index.js or simply got to the index.js file and run it.
(make sure localhost3003 is free).
